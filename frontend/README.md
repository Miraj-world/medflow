# MedFlow Frontend

React + Vite dashboard for the MedFlow care team experience.

## Local Run

1. Copy `frontend/.env.example` to `frontend/.env`.
2. Set `VITE_API_URL` to your backend API, for example `http://localhost:10000/api`.
3. Run `npm install`.
4. Run `npm run dev`.

## Pages

- Dashboard
- Patient List
- Patient Detail
- Analytics
